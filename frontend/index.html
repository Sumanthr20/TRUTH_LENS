System Overview

About TruthLens

TruthLens is an AI-based fake news detection system developed as a major academic project. It combines Natural Language Processing (NLP) and classical Machine Learning to analyse textual news content and classify it as credible or potentially misleading.

The system uses a TF-IDF vectoriser to convert text into feature vectors and a Logistic Regression classifier trained on a labelled dataset of real and fake news samples. Additional linguistic heuristics add an explainability layer, surfacing keywords and language patterns that commonly indicate misinformation.
